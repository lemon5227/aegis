Proposal: Aegis Governance V2 Update

Posted by @crypto_king 5h ago Governance
#Proposal #V2Upgrade #Active

This proposal outlines the new voting mechanism to prevent whale dominance within the Aegis ecosystem. We've observed that Quadratic Voting (QV) has been successful in other DAOs, and we propose implementing a modified version here.

The key changes involve:

  • Implementing a square root function for voting power.
  • Introducing a cooldown period for newly acquired tokens.
  • Adding a reputation-based multiplier for long-term holders.

We believe these changes will create a more equitable environment for all participants. Please review the attached technical specification document before casting your vote.
